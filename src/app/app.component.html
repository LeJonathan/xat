Guide

What do you want to do next with the bot?

> 1. Setup the bot for your server

The Setup process lets you:

  • Toggle whether or not you want xat IDs in people's names
  • Toggle whether or not you want toon names for new users
  • Set an announcement channel to receive new bot updates.
  • Create any roles required for powers to function (Blueman, Pink, Gagged, Banished, etc)

> 2. Acquire xats

The xat collecting process is straight forward, for every message you send you will be granted 1-5 xats on a cooldown (5-10 if you have days).

In addition to this, you can also use the !daily command for 100 xats (+25 per daily streak) every 23 hours.

Still not enough for you? The !work command can help boost your xat returns by 50-70 every 3 hours.

> 3. Buy powers

Powers transfer from server to server and work depending on if the server owner has properly setup the bot. To buy a power, simply !buy or !store to view the current selection.
Note: There are multiple pages which can be accessed through !store 2, 3, etc.

If there is a role associated to a power and it has been setup on your current server, you can use the !toggle command to enable it, i.e: !toggle Blueman

> 4. Flex your profile

You can create and manage your very own little xatspace, complete with customizations such as number of gifts you own, xats and current marriage/BFF using the !profile command.

To update your background, description or colors use !edit.

> 5. Explore the rest of the commands!

The xat bot attempts to re-create the xat experience on Discord, many of the commands will feel familiar if you've used the website! Scroll up and have a look - specifically at the "xat Simulator" section to see what's available to you.
